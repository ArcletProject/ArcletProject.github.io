概览 ​

Arclet Project 是包括 Alconna, Cithun, Letoderea, Entari, Tarina 在内的一系列与机器人开发框架相关、 但不局限于机器人开发的框架集合体

项目 ​

Arclet Project 目前有如下项目:

  • Alconna - 命令解析器

  • Cithun - 类文件系统的权限管理系统

  • Letoderea - 高效事件系统

  • Entari - 基于 Satori 跨平台协议的 IM 框架

  • Tarina - 功能与开发工具集合

  • NEPattern - 复杂正则匹配

  • Alconna-Tools - Alconna 拓展工具

  • Alconna-Graia - Alconna 与 Graia 的集成项目

  • nonebot-plugin-alconna - Alconna 与 Nonebot2 的集成项目

  • ArcletTS - TypeScript 版本的 Arclet 项目

参与开发 ​

我们非常欢迎各位有志之士能帮助我们以各种方式完善这个项目.

你可以通过以下几种方式参与进来:

  • 提交你的 Issue ,包括但不限于 bug 汇报, 新功能提案, 文档改进等.

  • 发起你的 Pull Requests ,直接将 想法 / 修复 合并到代码库中.

  • 参与 QQ 讨论 与开发者进行直接交流.
