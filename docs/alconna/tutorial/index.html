导语

介绍​

Alconna 是 Arclet Project 下的命令解析器

Alconna 由繁多的预设组件来架构其解析功能, 以用于更加精确的命令解析.

from arclet.alconna import Alconna

cmd = Alconna(["!", ".bot"], "获取{name}的涩图")
print(cmd.parse("!获取円香的涩图").header)

"{'name': '円香'}"

实时演示 (js)：

实时编辑器
let cmd = new Command("获取{name}的涩图", ["!", ".bot"], Args.push("count", Number, 1));


const Show = (props) => {
  let res = cmd.parse(props.input);
  return (
  <>
    <h2>
      匹配结果: {String(res.matched)}
    </h2>
    <p>
      正在获取 {res.mainArgs.get("count")} 张 {res.header.name} 的涩图...
    </p>
  </>
  )
};


render(<Show input="!获取円香的涩图 1"/>);

安装​

for Python

  • pdm
  • poetry
  • pip
pdm add arclet-alconna
poetry add arclet-alconna
pip install --upgrade arclet-alconna

for Nodejs

npm install @arcletjs/alconna

特点​

  • Alconna 既能解析纯文本, 也能解析如 MessageChain 的复杂数据, 或 List[str]等原始数据

  • Alconna 拥有强大的类型转换与繁多的类型预设, 可以自定义类型, 并且可以自定义类型的解析方式

  • Alconna 提供了简单的构造方法, 无需调整过多参数便可使用; 可以解析字符串与消息链

  • Alconna 会将解析的结果包装成 Arparma, 你可以通过 Arparma 获取传入的消息内容

  • Alconna 可以自定义解析器, 可以自定义解析器的解析方式, 可以自定义解析器的解析结果

  • Alconna 集成了模糊匹配、自定义语言文件等功能

性能参考​

在 i5-10210U 处理器上, Alconna 的性能大约为 41000~250000 msg/s, 取决于 Alconna 的复杂程度
